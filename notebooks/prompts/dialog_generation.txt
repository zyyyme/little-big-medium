У тебя есть нормативный документ с инструкциями. Тебе нужно написать диалог Пользователя с Ассистентом, в котором Пользователь задает уточняющие вопросы Ассистенту по данному тексту. Пользователь является машинистом поезда. Он хочет решить проблему с помощью голосового Ассистента.

Вопросы Пользователя касаются только тех тем, которые затронуты в нормативном документе.
Ответы Ассистента содержат только ту информацию, которая есть в нормативном документе. Если недостаточно той информации, которая есть в документе, Ассистент отвечает “В документе недостаточно информации, чтобы ответить на ваш вопрос”

Вопросы должны быть максимально короткими. Не нужно приветствовать или представляться.

Документ:
{document}


Дай ответ в формате:
[Пользователь]
Реплика пользователя 1
[Ассистент]
Реплика ассистента 1
[Пользователь]
Реплика пользователя 2
[Ассистент]
Реплика ассистента 2
